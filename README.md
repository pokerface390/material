## Descripción general

<p align="center">
Este portafolio digital desarrollado en Angular contiene una colección de 15 componentes que ejemplifican distintas funcionalidades y elementos de interfaz de usuario. Su objetivo es presentar mis habilidades técnicas mediante componentes funcionales, modernos y visualmente sobrios. Se utilizan colores neutros (blanco, gris, negro y azul) para proyectar claridad y profesionalismo.
</p>

## Justificación de diseño

<p align="center">
El diseño general está inspirado en interfaces tipo dashboard profesional. Las tarjetas con esquinas redondeadas y sombras suaves encapsulan cada componente, facilitando su comprensión. Se ha priorizado el uso de colores neutros con acentos azules para facilitar la lectura y transmitir seriedad visual.
</p>

### Diseño técnico

<p align="center">
  <strong>Paleta:</strong> Blanco, gris, negro y azul<br>
  <strong>Tipografía:</strong> Sans-serif moderna, clara y legible<br>
  <strong>Layout:</strong> Grid responsivo que se adapta a cualquier dispositivo<br>
  <strong>Accesibilidad:</strong> Contraste alto, feedback visual y componentes reutilizables
</p>
